# Platform v2.0 Capability Matrix

This document provides a comprehensive mapping of all registered Model Context Protocol (MCP) tools exposed by **Platform v2.0**, classified by architectural subsystem, scope, and capabilities.

---

## 🛠️ MCP Tool Classifications

The platform registers **100+ fine-grained tools** to provide LLM clients with direct access to Windows operating system resources, multi-agent frameworks, and distributed clusters.

| Subsystem | Tool Name | Scope / Namespace | Capabilities |
| :--- | :--- | :--- | :--- |
